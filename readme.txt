SMS computer control (Режим PDU)

Получается три программы

1. Исключительно для отправки SMS - получает на вход номер и само сообщение. Используется для скриптов и прочих оповещений

2. Сервис windows который висит и с периодичностью раз в 30 секунд (настраивамый интервал от 10 до 600 секунд), проверяет входящие СМС и список преднастроенных команд, если есть соответвствие с текстом, паролем и командой, то команда выполняется. 

3. Программа-конфигуратор для сервиса (не обязательна, можно конфигурировать вручную, но через программу удобнее и более наглядно, плюс можно сразу же проверить выполнение команды)


а) Входящий номер (если пусто, то с любого), может быть указано несколько, через запятую, номер одинадцать символов с первой семёркой
б) Команда (здесь можно указывать имя команды (к примеру restart или reboot) при получении смс с такой командой будет выполнятся сценарий
в) Путь к выполняемому сценарию или exe файлу.
